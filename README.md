# PiDrone

![logo2](https://raw.githubusercontent.com/keiya/PiDrone/master/logo2.svg)

## Concept
* WebSocket based controll link
  * Control Uplink
  * Telemetry Downlink
    * Barometer
    * Attitude (Main & Backup)
    * GPS Position & Altitude
    * Input Voltage
    * Raspberry Pi CPU Temperature & Clock speed
